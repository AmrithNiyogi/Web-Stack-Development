Introduction to Canvas in HTML5

The <canvas> element in HTML5 is used to draw graphics on a web page using JavaScript. It provides a resolution-dependent bitmap canvas, which can be used to render graphs, game graphics, and other visual images on the fly.

Basic Usage

The <canvas> element itself is just a container for the graphics; you must use JavaScript to actually draw the graphics.

<canvas id="myCanvas" width="500" height="300">
  Your browser does not support the canvas element.
</canvas>

<script>
  var canvas = document.getElementById('myCanvas');
  var context = canvas.getContext('2d');
  context.fillStyle = '#FF0000';
  context.fillRect(0, 0, 150, 75);
</script>

Canvas Context

The getContext() method returns an object that provides methods and properties for drawing on the canvas. The "2d" context allows for two-dimensional drawing, while "webgl" allows for three-dimensional drawing.

Drawing Shapes

Canvas provides methods for drawing shapes, including rectangles, circles, and paths:

  • fillRect(x, y, width, height): Draws a filled rectangle.
  • strokeRect(x, y, width, height): Draws a rectangular outline.
  • clearRect(x, y, width, height): Clears the specified rectangular area.
  • beginPath(): Begins a new path.
  • arc(x, y, radius, startAngle, endAngle, anticlockwise): Draws a circle or arc.
  • lineTo(x, y): Draws a line from the current drawing position to the specified position.
  • moveTo(x, y): Moves the path to the specified point in the canvas, without creating a line.
  • stroke(): Draws the outline of the current path.
  • fill(): Fills the current path.

Drawing Text

Canvas provides methods for rendering text:

  • fillText(text, x, y): Draws filled text at the specified coordinates.
  • strokeText(text, x, y): Draws text outlines at the specified coordinates.
var canvas = document.getElementById('myCanvas');
var context = canvas.getContext('2d');
context.font = '30px Arial';
context.fillText('Hello World', 10, 50);
context.strokeText('Hello World', 10, 100);

Gradients and Patterns

Canvas supports gradients and patterns for more complex drawings:

  • createLinearGradient(x0, y0, x1, y1): Creates a linear gradient.
  • createRadialGradient(x0, y0, r0, x1, y1, r1): Creates a radial gradient.
  • createPattern(image, repetition): Creates a pattern using the specified image.

Advanced Drawing

Canvas allows for more advanced drawing techniques, including transformations and image manipulation:

  • rotate(angle): Rotates the canvas around the current origin.
  • scale(x, y): Scales the canvas units by the specified factors.
  • translate(x, y): Moves the canvas and its origin on the grid.
  • drawImage(image, dx, dy): Draws an image onto the canvas.

Example: Drawing a Circle

Here is an example of how to draw a simple circle on the canvas:

<canvas id="circleCanvas" width="200" height="200"></canvas>

<script>
  var canvas = document.getElementById('circleCanvas');
  var context = canvas.getContext('2d');
  context.beginPath();
  context.arc(100, 100, 50, 0, 2 * Math.PI);
  context.stroke();
</script>

Conclusion

The <canvas> element in HTML5 provides a powerful and flexible way to draw graphics directly in the browser. With a wide range of methods and properties, developers can create dynamic and interactive graphics for games, data visualization, and other applications.
